## 📁 Struktur Project

Berikut struktur file pada project **SaifulCloud Center**:

SaifulCloud-Center/  
│  
├── .gitignore                  → Mengatur file/folder yang tidak diupload ke GitHub (cache, env, dll)  
├── SaifulCloud_Center.ipynb    → Notebook utama untuk menjalankan aplikasi (EC2 & S3 management)  
├── requirements.txt            → Daftar library Python yang dibutuhkan  
├── manual penggunaan.pdf       → Dokumentasi panduan penggunaan aplikasi  
├── EC2_Compute_Saiful_amri.png → Screenshot fitur Amazon EC2  
├── S3_Bucket_Saiful_amri.png   → Screenshot fitur Amazon S3  
└── README.md                   → Dokumentasi utama project  

---

### 📌 Penjelasan Struktur

- **.gitignore**  
  File ini digunakan untuk menentukan file/folder yang tidak perlu diupload ke repository GitHub, seperti file cache, virtual environment, atau file sementara.

- **SaifulCloud_Center.ipynb**  
  File utama berbasis Jupyter Notebook yang berisi seluruh kode program untuk mengelola AWS EC2 dan S3.

- **requirements.txt**  
  Berisi daftar dependency seperti `boto3`, `ipywidgets`, dll yang harus diinstall sebelum menjalankan aplikasi.

- **manual penggunaan.pdf**  
  Panduan penggunaan aplikasi dalam bentuk dokumen PDF agar user lebih mudah memahami langkah-langkahnya.

- **EC2_Compute_Saiful_amri.png**  
  Gambar hasil tampilan fitur EC2 seperti launch instance, terminate, dll.

- **S3_Bucket_Saiful_amri.png**  
  Gambar hasil tampilan fitur S3 seperti create bucket dan delete bucket.

- **README.md**  
  File dokumentasi utama yang menjelaskan keseluruhan project.
